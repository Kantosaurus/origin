Signature novel mechanisms (the "beats peers" core)
① CAS-deduped stage handoff. Naive multi-agent pipelines re-send the whole transcript to each next agent (huge token cost). Here each stage emits a structured report whose artifacts (spec, diff, test log) are CAS handles; the next agent gets handles + a compact summary and lazily rehydrates only what it needs. Shared context is stored once, referenced — not recopied. Token win vs cline/kilocode/opencode.
② Board = CRDT op-log. Every stage transition + every human drag is an op (origin-plan style). Real-time multi-client sync is free and conflict-free; the op-log is also the audit trail + replay. vs peers' DB + last-write-wins websocket broadcast.
③ Per-stage sandbox profiles. Each stage's agent gets a tool allow-list matching its role — Plan/Design are read-only (literally cannot write code), Build edits only inside its worktree, Review can run gh but not force-push. Uses origin's permission-gated tools + per-platform sandbox.
